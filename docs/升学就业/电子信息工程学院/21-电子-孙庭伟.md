# 关于保研
* 我想通过我的了解和感受去帮你感知到更多**宏观层面**需要考虑的点。
* 首先明确一点，在升学层面，社会资源分配是不均的。读研而言，安徽省内211的学生就是比相似情况的其他院校学生更容易进入科大。
* 保研需要衡量的要素大致有：学校title，课题组方向，导师人品，城市地域，预估就读体验等。权重你来分配。我觉得导师比较重要。
* 尽量明晰你对读研的需求，不要辛苦了三年，不知道为什么要保研。是明确想读研就业，还是明确想读博深造，还是觉得都要先读研，读博另说。
关于就业：可以查看目标课题组往届学长学姐的就业去向，如果你去，大概率也是类似去向。很想具体了解，可通过学生邮箱多多礼貌打听，赠送一杯奶茶等。
具体的就业选择感兴趣可以查一查，网上很多。以意向转码为例，有无实习就是很重要，学校title相对不太重要，这个时候导师是否放实习就需要你重点关注。
关于读博：一些好的学校可能给你直博名额，但慎重直博，直博是真的需要静下来做学术的。读博的话，课题组的方向实力，导师的人品更重要，还要考虑家庭经济条件。
* 你远大于你的大学。这个读研选择能不能给你带来生活其它方面的突破和改变？还是会抑制你寻找生活更多精彩的可能？专业是理工科的你又对读研/读博这段年华的期待是什么样的？
* 读研辛苦是难以避免的，但是什么样的累是更愿意接受的？
  
  

# 夏令营
夏令营的时候用的还是前五学期排名(6/184)，所以其实还是挺吃亏的(尽管教务处打印出来还是并列第4，很残酷但确实需要排清楚哈哈)。
夏令营投递时期几乎还是只能用到前五学期绩点排名，若想夏令营期间拿到不错的offer，前五学期绩点排名>前六学期绩点排名>前六学期综合排名。
另外有一个很主观的感受：能专业第一就不要专业第二，能专业前三就一定要冲前三。考虑到排名和论文情况，我就只好多多尝试了。
面试时间撞了挺多的，很多机会都白白浪费了，还有就是后来才发觉，很多营都可以进但是没有报。提醒大家，相信一点点缘分，勇敢一点。
## 夏令营相关面试
* 武汉大学+电子信息工程学院+电子信息(通信)专硕（7.4）
>这个面试我有点意外的，感觉是论文和六级给背景分添加不少(很多高校进营都会有一套背景分计算系统)。后面才了解"等级森严"到：211只给专硕。
第一场面试也是最终去向，有点意思哈哈哈。面试总分=笔试+面试+背景分。当天上午宣讲参观，下午笔试，后一天面试。笔试范围有通知说明(可以刷刷期末和考研题)
连续两年有变。面试主要分为两场即两间会议室，每间六位老师，5分钟。第一间自己从四门专业课(信号+通信，模电，电磁场与电磁波，编译原理)
选择两门进行回答，(然而去年信号和通信是分开的，我就只考虑复习这两门，电磁场只看了一点点)我就只好选信号+通信和电磁场了。结果首先是提问的电磁场，
第二个问题就开始支支吾吾了，我就厚着脸皮(笑)和老师说学生想从事信号方向，老师可以多从这方面考察学生嘛？那个男老师有点小尴尬(悲)，随即有老师出来"解救"我，开始逐位继续提问。
因为这方面复习很充分，从头库库回答到结束(时间飞快)，当时就松了一口气。第二个房间是英文自我介绍和项目提问。自我介绍很熟了，说一半就让停，熟悉好自己的项目，
项目提问也顺利通过。总体感觉老师们都很温和与包容(怕我尴尬，第一个信号问题都直接问什么是线性系统了)，心里也比较有底了。最后专硕优营排名10/90，想着有学上了，
后续面试也就完全放开自己了。

* 南开大学+光电信息科学与工程学院+光电信息工程（7.8）
>南开夏令营又叫"飞书计划"，需要进群，提前选择并完成导师发布任务，难度相对适中。南开光电学院的夏令营体验是我觉得最好的，包吃包住
>研究生学长学姐们也非常有耐心地解答疑惑。对于面试，我实在是有点跨专业的嫌疑(本是对老师感兴趣，后续反思还是应该报隔壁电子通信系)
>所以面试提问很尴尬：信号高数之外，光电专业知识确实不会，但整体面试氛围还是很友善的，总时约20分钟。最后也获得了优营。考虑到离家太远且就业问题
>就也不太想去了。

* 成都电子科技大学+信息与通信工程学院+电子信息(新一代)专硕（7.15）
>成电我是提前联系了老师，属于课题组私下面试。同时今年学院也尝试新开了夏令营，可以留意一下。面试是群面，总时约40分钟。首先是PPT展示，然后是英文提问+中文提问。
>英文提问了我论文概述和相关问题。中文提问涵盖了概率论信号和通信，还是相对有点细的，例如为什么s变换大多考虑右半轴，存在左半轴的s变换吗？老师还会问你一些比较宏观的问题，
>类似未来研究选择，拿到一个课题你该如何下手等。最后也收到了offer。后续非常惊喜地认识了组内的安大学长，和学长细细交流后发现组内情况可能并不是我想象那样，就也不再考虑了。

# 预推免
由于夏令营已经有几个确定的offer了，综合考虑到排名提升(4/184)和预推免难度的变化，以及倾向于不读博+好title+好地区，就做了如下选择：
浙大电院太难且需要直博，UIUC难度也大了起来且导师方向不太契合本科经历，索性就还报浙大工程师宁波方向。
复旦的话，和学长交流后联系老师可能会有机会，但也需要直博，就尝试了工程硕士。
南大，还是电院电子信息专硕。不意外的，上述全都落空emm。  
**后悔的选择**：没有尝试同济电院和北航2系。北航另说，同济电院感觉今年真挺有机会的，主要是不想去笔试hhh。
我准备的重心就是科大了。很多个夜晚翻找着科大6系的招生导师名单，一个个联系。已读不回十分沮丧。
因为夏令营排名不是第四，感觉有论文也不好冲，索性就报了先研院，难受的是先研院今年也缩招了，招电子通信方向的老师少了很多，那时还没考虑网安。后续了解到我们院今年报科大6系的人比以往少了一些，
我可能还是有机会进营的。**强调一下，夏令营期间大家是一起努力的战友，了解一下自己同学夏令营的投报情况，可能利于彼此做出更勇敢的选择**。预推免也考虑过报网安学院和先研院(预推免只能报一个学院，也为后续埋下伏笔)，但还是惯性思维报了六系。  
当然，还是十分建议各位小伙伴，如果夏令营没有取得心仪的offer，这段时间都还是不要太掉以轻心，也不要太焦虑。可以把精力分配在信息搜集上，真的强调要多套磁老师！这段时间往后，会有大量的offer释放出来，不要觉得不好意思，展示自己是自己的课题，
是否接纳是老师的课题，可能老师当时会已读不回，但如果老师后续被鸽，又需要招生，重新翻邮箱会有新的可能！另外结合今年预推免的情况看，要留心往届安大学长在群里分享的课题组的招生机会！上岸率非常高。

# 关于选择
尽量明晰你对读研的需求，不要辛苦了三年，不知道为什么要保研。是明确想读研就业，还是明确想读博深造，还是觉得都要先读研，读博另说。  
关于就业：可以查看目标课题组往届学长学姐的就业去向，如果你去，大概率也是类似去向。很想具体了解，可通过学生邮箱多多礼貌打听，赠送一杯奶茶等。
具体的就业选择感兴趣可以查一查，网上很多。  
以意向转码为例，有无实习就是很重要，学校title相对不太重要，这个时候导师是否放实习就需要你重点关注。  
关于读博：一些好的学校可能给你直博名额，但慎重直博，直博是真的需要静下来做学术的。读博的话，课题组的方向实力，导师的人品更重要，还要考虑家庭经济条件。  
**借用一位优秀学长的话来说：“我远远大于我的大学，我始终相信，人生是一个长期的均值回归，人生各个阶段的升职也好求职也好，这只是这条长达几十年均值回归上的一点点小噪音，
你最后一定会和你的付出、能力与野心匹配上，所以我们不必为了一些短期的噪音而过多的焦虑。”**
>还是那句话，在缘分努力挣扎之后，你所做出的选择就是当下能做出的最好的选择了，适合自己的最重要！没有正确的选择，去让它变成正确的吧。
